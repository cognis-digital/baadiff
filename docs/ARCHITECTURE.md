# BAADIFF — Architecture

> Scan a repo or infra manifest for HIPAA Security Rule gaps and produce a Business Associate readiness scorecard.

```
input ──▶ collect ──▶ rules/analyzers ──▶ score ──▶ findings ──▶ table · json
                              │                          │
                         (this repo)                 MCP tool (agents)
```

- **collect** normalizes the target (file/dir/API) into records.
- **rules/analyzers** apply the heuristics shipped in `baadiff/core.py`.
- **score** ranks by severity.
- **MCP server** (`baadiff mcp`) exposes `scan` for Cognis.Studio agents.

Extend by adding a rule + a test + a `demos/NN-*/SCENARIO.md`. See [CONTRIBUTING.md](../CONTRIBUTING.md).
